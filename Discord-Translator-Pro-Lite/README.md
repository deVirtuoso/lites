# Discord Translator Lite

Minimal Chrome extension that translates Discord messages without the account,
billing, upgrade, trial, locale, or bundled React screens from the Pro package.

## Install unpacked

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this folder

## Kept functionality

- Auto-translate visible Discord messages
- Optional bilingual display
- Optional outgoing composer translation button
- Microsoft, Yandex, Google, and Google API-key-backed translation fallback
- Local popup settings only

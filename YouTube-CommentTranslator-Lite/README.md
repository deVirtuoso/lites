# YT Chat Translator Lite

Chrome extension that auto-translates YouTube video comments and live chat.

## Install (unpacked)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this folder

## Features

- Auto-translate comments and live chat
- Microsoft, Yandex, or Google translation engines
- Toggle between translation-only and bilingual display (popup or on-page button)

## Settings

Use the extension popup to set your language, engine, and colors. Use **Both languages** / the floating button on YouTube to show or hide originals.
